# ATELIER 8 – Circular Luxury Dashboard (Colourful Version)

This Streamlit app is a **storytelling dashboard** for ATELIER 8, a circular luxury restoration & authentication studio.

Key features:
- Clean 4-tab layout: Overview, Customer Segments, Predictive Models, Upload & Score.
- Colourful Plotly charts (bar, histogram, box, scatter, and pie charts).
- Plain-English explanations under each visual so that even a non-technical viewer understands the insight.

## Files

- `app.py` – Main Streamlit application.
- `atelier8_survey_data.csv` – Synthetic survey dataset (400 respondents).
- `requirements.txt` – Python dependencies for GitHub/Streamlit deployment.

## How to run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

Then upload this repo to GitHub and connect it to **Streamlit Community Cloud**.
